Using the Layered Book Foldable TM by Dinah Zike Vocabulary Application Have students create this Foldable to help them review vocabulary words. Have them write a word on each tab and then ﬂip the tab to write the deﬁnition. The same thing can be done with antonyms and synonyms. Phonics/Spelling Application A review/study guide of letter sounds and word parts can be done with this Foldable. For example: • Vowels • Preﬁxes and sufﬁxes • Base words • Digraphs (th, wh, sh, ch) Comprehension Application Use the Foldable to aid in the following skills reinforcement: • Character study (one tab per story character) • Summarize/Retelling • Generating and asking questions Study Skills and Grammar Applications This Foldable can be used to review/reinforce concepts studied. 22 Foldables
